3d Models folder has the models that we printed to house our electronics and to be the body of the blimp.

.py files are main programs that the blimp runs, they are started on startup with the exception of the takepic.py and takepics.py as those are only ran when called.

.txt files are information that we want to save between power cycles as if blimp turns off we want it to restart with the same information vs losing all information if we made it a queue within the files.
